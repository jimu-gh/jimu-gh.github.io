• Personal live feed of current state ballot measures
• User Groups/Endorsements system with quorum feature
• Real-time Socket.IO chat (global and groups)
With PixelTune, I independently conceived and created an iOS camera app that suggests music recommendations based on image recognition from Google Cloud Vision. I've always been fascinated with the synaesthetic aspect of how 1's and 0's can make us "feel".
• Support for emotion recognition: joy, sadness, sorrow, surprise
• Spotify SDK user authentication (Facebook support)
• In-app streaming using Spotify Web API
In creating this Augmented Reality game, I wanted to experiment with iOS CoreLocation to make something easy and fun to play while maintaining replay value. In HomeSafe, the player's goal is to reach a target location on the map (by actually walking) without running into virtual monsters with simple AI.
• Uses live GPS to simulate user and monster location
• "Smart" and random monster pathing
• Conceived and created in a single day
